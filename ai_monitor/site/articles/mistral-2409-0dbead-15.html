Mistral March Releases: Voxtral TTS, Mistral Small 4, Leanstral, Moderation 2603

Mistral shipped four major releases in March 2026, spanning a text-to-speech model, a hybrid instruct-reasoning-coder, a formal proof agent, and an updated moderation model.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a state-of-the-art text-to-speech model supporting zero-shot voice cloning, multilingual output, and real-time streaming. The release marks Mistral's most capable entry yet in the audio synthesis space, directly competing with established TTS providers.

March 16 brought Mistral Small 4 (mistral-small-2603), a hybrid model that unifies instruct, reasoning, and coding capabilities in a single multimodal package with a 256k-token context window. The same day saw the release of Leanstral (labs-leanstral-2603), Mistral's first open-source code agent purpose-built for Lean 4 formal proof engineering—a niche but technically demanding area.

Mistral Moderation 2603 (mistral-moderation-2603) arrived March 12, and the month also added Custom Guardrails support for Agents and Conversations, configurable via the guardrails parameter on agents or per-request on both the /v1/conversations and /v1/chat/completions endpoints.

Why this matters

  • Voxtral TTS gives developers a new zero-shot voice cloning option for real-time audio generation, a capability that previously required specialized vendors.
  • Mistral Small 4's 256k context window and hybrid instruct-reasoning-coder design targets developers who need a single model for complex, long-document tasks.
  • Leanstral's open-source formal proof agent fills a gap for developers working in mathematics and software verification using the Lean 4 language.
