Mistral March 2026 Releases: Voxtral TTS, Mistral Small 4, Leanstral, and More

Mistral released four notable updates in March 2026 spanning text-to-speech, a hybrid instruct/reasoning model, a Lean 4 code agent, and a new moderation model.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

On March 23, Mistral published Voxtral TTS (voxtral-tts-2603), a text-to-speech model that supports zero-shot voice cloning, multilingual output, and real-time streaming. This marks Mistral's entry into the voice synthesis space, positioning the model alongside its existing audio transcription offerings.

March 16 brought two releases. Mistral Small 4 (mistral-small-2603) is a hybrid model that combines instruct, reasoning, and coding capabilities in a single multimodal package, backed by a 256k context window. Also that day, Mistral released Leanstral (labs-leanstral-2603), described as the company's first open-source code agent purpose-built for Lean 4 formal proof engineering.

On March 12, Mistral shipped Mistral Moderation 2603 (mistral-moderation-2603). Separately, Custom Guardrails support was added for the Agents API, allowing guardrails to be configured directly on an Agent object, per-request on POST /v1/conversations, and per-request on POST /v1/chat/completions.

Why this matters

  • Voxtral TTS adds a zero-shot voice cloning and real-time streaming TTS capability to Mistral's growing audio portfolio, which already includes Voxtral Small, Voxtral Mini, and Voxtral Mini Transcribe models.
  • Mistral Small 4 unifies instruct, reasoning, and coding in one 256k-context model, simplifying the stack for developers who currently use separate models for those tasks.
  • Leanstral is a niche but notable release — it targets formal verification engineers working in Lean 4, a proof assistant used in academic and safety-critical software contexts.
