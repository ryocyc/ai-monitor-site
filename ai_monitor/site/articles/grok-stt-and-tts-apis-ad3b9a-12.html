xAI Launches Grok Speech-to-Text and Text-to-Speech APIs

xAI has released two standalone audio APIs — Grok Speech to Text and Grok Text to Speech — built on the infrastructure powering Grok Voice in Tesla vehicles and Starlink customer support.

xAI Blog 2026-04-22 23:49 UTC Key: xai-blog::identity::grok-stt-and-tts-apis Confidence: strong Mode: claude

Article body

The Grok Speech to Text API offers both a batch REST endpoint and a real-time WebSocket endpoint for low-latency transcription. It ships with word-level timestamps, speaker diarization, multichannel audio support, and Inverse Text Normalization that converts raw spoken output into structured text — turning spoken numbers, dates, and currencies into their proper formatted equivalents. The API claims an overall word error rate of 6.9% in internal benchmarks, compared against ElevenLabs (9.0%), Deepgram (11.0%), and AssemblyAI (12.9%), with performance evaluated across phone calls, meetings, video/podcasts, and telephony. It supports 25 or more languages with seamless in-call language switching.

Grok Text to Speech targets developers building voice agents, podcasts, accessibility tools, and interactive audio. It provides both REST and WebSocket endpoints and includes a set of inline speech tags — `\u005Blaugh\u005D`, `\u005Bsigh\u005D`, `\u005Bwhisper\u005D` — that let creators add prosody and emotional inflection without complex markup. An online voice playground lets developers audition voices before integrating.

Pricing is straightforward and flat-rate. Speech to Text costs $0.10 per audio hour for batch transcription and $0.20 per audio hour for streaming. Text to Speech is priced at $4.20 per one million characters. Both are available now through the xAI API console and developer documentation.

Why this matters

  • xAI is directly competing with established speech-API providers (Deepgram, AssemblyAI, ElevenLabs) and claims a meaningfully lower word error rate across multiple domains.
  • The TTS speech-tags feature — adding `\u005Blaugh\u005D`, `\u005Bsigh\u005D`, or `\u005Bwhisper\u005D` directly in text — is an unusually developer-friendly control surface not widely offered by competitors.
  • By tying the new APIs to the Grok Voice stack already deployed in Tesla and Starlink, xAI signals real-world production validation at scale beyond typical API launches.
