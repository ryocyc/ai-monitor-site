Mistral Ships Four March Releases Spanning TTS, Reasoning, and Formal Proofs

Mistral released four distinct products in March 2026, including its first text-to-speech model, an updated small flagship, an open-source formal proof agent, and a new moderation model.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: moderate Mode: claude

Article body

Mistral's most active March release window brought Voxtral TTS (voxtral-tts-2603) on March 23, described as a state-of-the-art text-to-speech model supporting zero-shot voice cloning, multilingual output, and real-time streaming. This marks Mistral's entry into the speech synthesis space, giving developers a single API for generating audio without needing a separate vendor.

On March 16, Mistral published Mistral Small 4 (mistral-small-2603), a hybrid model that consolidates instruct, reasoning, and coding capabilities into one multimodal package with a 256k token context window. The same day also saw the debut of Leanstral (labs-leanstral-2603), Mistral's first open-source code agent purpose-built for Lean 4 formal proof engineering — a niche but growing area of interest for mathematical and verification-focused workflows.

March 12 rounded out the month with Mistral Moderation 2603 (mistral-moderation-2603). The changelog also flags a broader API update on Custom Guardrails, allowing configuration directly on an Agent via a guardrails parameter, as well as per-request control on both the /v1/conversations and /v1/chat/completions endpoints.

Why this matters

  • Voxtral TTS marks Mistral's first entry into speech synthesis, competing directly with providers like ElevenLabs and OpenAI's audio APIs.
  • Mistral Small 4's hybrid approach merges instruct, reasoning, and coding in one model with a 256k context, signaling a push toward unified small-model utility.
  • Leanstral's focus on Lean 4 formal proof engineering is a direct appeal to academic and verification-focused developers, an underserved niche in the LLM ecosystem.

  • All details are drawn from Mistral's public changelog at docs.mistral.ai/getting-started/changelog/. The changelog provides model names, version identifiers, and brief capability descriptions but does not include technical benchmarks, pricing, or availability timelines for the March releases.
