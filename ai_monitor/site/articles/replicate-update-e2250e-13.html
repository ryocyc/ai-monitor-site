Replicate Publishes Agent Skills for Claude Code, Codex, and OpenCode

Replicate now offers agent skills — markdown instruction files that give coding assistants specialized knowledge for working with its AI models.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

Replicate has published a collection of agent skills: markdown files that help coding assistants like Claude Code, OpenAI Codex, and OpenCode work more effectively with AI models on its platform. The skills cover model discovery, API execution, and prompting strategies for image and video generation models.

The skills follow the open Agent Skills spec and can be installed with a single command: `npx skills add replicate/skills`. Once installed, they are automatically configured for use with compatible coding assistants.

Replicate frames the skills as a complement to its existing MCP server. While the MCP server exposes API tools to a coding assistant, agent skills supply the knowledge to use those tools well — including guidance on model selection, prompt writing, and tradeoffs to consider. Details are available in the agent skills reference and the GitHub repository.

Why this matters

  • Coding assistants can now reason better about which Replicate model to use for a given task, rather than guessing blindly.
  • The skills work across multiple tools — Claude Code, OpenAI Codex, and OpenCode — making the ecosystem more consistent.
  • Because the skills follow an open spec, developers can inspect, fork, or extend them independently of Replicate's platform.
