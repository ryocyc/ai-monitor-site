AWS Launches Frontier Agents for Security Testing and Cloud Operations

AWS Security Agent and AWS DevOps Agent, both introduced as frontier agents at re:Invent, have exited preview and are now generally available to customers.

Article body

AWS has made two autonomous AI agents—AWS Security Agent and AWS DevOps Agent—generally available, marking a key milestone for the frontier agents class of capabilities the company unveiled at re:Invent. These agents work independently to achieve defined goals, scale across concurrent tasks, and run persistently for hours or days without constant human oversight, fundamentally shifting how organizations handle security testing and cloud operations at scale.

Why this matters

  • Autonomous penetration testing is now commercially available from a major cloud provider, lowering the barrier to continuous security validation.
  • The general availability signals that frontier agents—a new AI capability category—are moving from preview concepts to production-ready products.
  • Both agents run for extended periods without human intervention, representing a step beyond single-prompt AI interactions in enterprise DevOps and security workflows.
