SGLang v0.5.10.post1: Piecewise CUDA Graph Now Default, Elastic EP Integrated

The SGLang LLM inference framework ships a patch release defaulting Piecewise CUDA Graph execution and integrating Elastic NIXL-EP for partial failure tolerance on DeepSeek MoE deployments.

GitHub SGLang Releases 2026-04-22 15:01 UTC Key: github-sglang-releases::identity::github-sglang-releases-sglang-release-activ Confidence: strong Mode: claude

Article body

SGLang v0.5.10.post1 patches the upstream v0.5.10 release with a targeted fix to FlashInfer's JIT cubin downloader, bumping the dependency from v0.6.7.post2 to v0.6.7.post3. The full v0.5.10 release that precedes it is substantial in its own right, making Piecewise CUDA Graph the default execution mode for the framework. The change is designed to cut memory overhead and lift throughput specifically for models that exercise complex control flow patterns.

The release also integrates Elastic NIXL-EP into SGLang, bringing partial failure tolerance to DeepSeek MoE deployments. In practice, if a GPU fails mid-workload, expert weights are redistributed across surviving GPUs and inference continues without a full cluster restart. Also notable is a GPU staging buffer for prefill-decode disaggregation that gathers scattered head slices into contiguous memory for bulk RDMA transfer, cutting RDMA request counts on GQA models by approximately 1,000x.

Broader changes in v0.5.10 span the stack: the official Flash Attention 4 library replaces the prior flash-attn package, adding Blackwell GPU support; transformers is bumped from 4.57.1 to 5.3.0, unlocking GLM-5 support; a native MLX backend enables SGLang to run inference directly on Apple Silicon without CUDA; and new model support covers GLM-5, Nemotron-3-Super, Mistral Small 4, Voxtral, and multiple diffusion models including LTX-2 and Hunyuan3D-2. A CVE fix (CVE-2026-3989) replaces unsafe pickle.loads with SafeUnpickler in replay_request_dump.py.

Why this matters

  • Piecewise CUDA Graph as default directly improves throughput and memory efficiency for structured and branching model architectures, which are increasingly common in reasoning and agentic workflows.
  • Elastic EP for partial failure tolerance removes a key operational risk in distributed DeepSeek MoE serving—workloads can now survive individual GPU failures without restarting, improving reliability in production.
  • The transformers 5.3.0 upgrade, native Apple Silicon support via MLX, and expanded diffusion model coverage signal SGLang's broadening scope beyond pure LLM inference into multimodal and edge deployments.
