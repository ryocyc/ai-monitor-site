Replicate Publishes Agent Skills for Claude Code, Codex, and OpenCode

Replicate now offers markdown instruction files called agent skills that give coding assistants expert-level knowledge of its model catalog, API, and prompting techniques.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: moderate Mode: claude

Article body

Replicate has published a collection of agent skills — markdown instruction files designed to give AI coding assistants specialized knowledge about working with its platform. The skills cover model discovery, model comparison, and API execution, along with detailed prompting guidance for image generation and video generation models. They follow the open Agent Skills spec and are compatible with Claude Code, OpenCode, OpenAI Codex, and other tools that support the spec.

The skills are installed via a single npm command that pulls all available instructions into a project and configures them for the coding assistant automatically. Replicate positions the skills as a complement to its existing MCP server: while the MCP server provides the actual API tools, the skills supply the know-how to use those tools effectively — including which models to pick for a given task, how to write prompts for them, and what tradeoffs to expect.

The changelog also notes several other recent additions to the platform, including a fallback model option for Nano Banana Pro that routes to ByteDance Seedream 5.0 lite when Google's API is at capacity, and a filter on the predictions API to show only results created through the web interface. Replicate's MCP server also gained automatic discoverability through the official MCP Registry, accessible via a standard /.well-known/mcp/server.json endpoint.

Why this matters

  • Agent skills turn Replicate into a first-class partner for AI coding tools, reducing the trial-and-error needed to pick the right model or write effective prompts.
  • The open Agent Skills spec means these instructions are portable — developers using Claude Code, Codex, or OpenCode all benefit from the same curated knowledge.
  • Pairing MCP's API tools with the new skills gives coding assistants a full stack of both capabilities and judgment, potentially automating more of the model-selection workflow end-to-end.
