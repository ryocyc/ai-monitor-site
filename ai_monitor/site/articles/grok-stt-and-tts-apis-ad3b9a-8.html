xAI Launches Grok Speech-to-Text and Text-to-Speech APIs

xAI has released two standalone audio APIs — Grok Speech to Text and Grok Text to Speech — promising high accuracy, low latency, and multilingual support for developers building voice-enabled applications.

xAI Blog 2026-04-22 23:49 UTC Key: xai-blog::identity::grok-stt-and-tts-apis Confidence: strong Mode: claude

Article body

On April 17, 2026, xAI announced two new standalone audio APIs: Grok Speech to Text (STT) and Grok Text to Speech (TTS). The company says both are built on the same underlying stack used by Grok Voice, Tesla vehicles, and Starlink customer support. Developers can access the STT API via REST for batch transcription or WebSocket for real-time streaming, and the TTS API the same way. Both products are designed to be integrated into voice agents, transcription tools, accessibility applications, podcasts, and interactive audio experiences.

Grok STT's benchmark table claims an overall word error rate of 6.9%, compared against ElevenLabs (9.0%), Deepgram (11.0%), and AssemblyAI (12.9%) across phone call entities, video/podcasts, meetings, and telephone domains. The API also includes word-level timestamps, speaker diarization, multichannel audio support, and Inverse Text Normalization that formats raw spoken numbers, dates, and currencies into structured output. Grok STT supports 25+ languages. Pricing is straightforward: $0.10 per hour for batch processing and $0.20 per hour for streaming.

Grok TTS focuses on natural, expressive voice output controlled via inline speech tags such as [laugh], [sigh], [whisper], and others for prosody and emotion. Developers can use either the REST API for long-form text or the WebSocket API for real-time speech generation. Pricing is $4.20 per one million characters, with no hidden fees. Both APIs are accessible through the xAI documentation, API console, and a voice playground.

Why this matters

  • xAI is directly competing with established speech API providers — including Deepgram, AssemblyAI, and ElevenLabs — and its benchmark figures position Grok STT as a potentially superior option for enterprise transcription workloads.
  • The pricing model is transparent and tiered: STT is charged by the hour ($0.10–$0.20) and TTS by the million characters ($4.20), with no tiered or hidden costs, making budgeting predictable for developers at any scale.
  • By exposing the same audio stack used in Tesla vehicles and Starlink support, xAI signals that these APIs are production-hardened from day one, not experimental previews.
