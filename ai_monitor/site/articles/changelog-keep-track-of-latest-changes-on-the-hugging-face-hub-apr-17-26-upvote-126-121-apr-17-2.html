Hugging Face Adds Auto-Generated /agents.md Endpoints to All Gradio Spaces

Hugging Face now gives every Gradio Space a machine-readable API description at /agents.md, letting AI coding agents like Claude Code and Codex interact with Spaces without manual setup.

Hugging Face Changelog 2026-04-25 07:00 UTC Key: hugging-face-changelog::merge::docshugging-face-changelogchangelog-keep-track-of-latest-changes-on-the-hugging-face-hub-apr-17-26-upvote-126-121-apr-17-26-spaces-agents-md-for-your-coding-age-changelog-keep-track-of-latest-changes-on-the Confidence: strong Mode: claude

Article body

Hugging Face announced on April 17, 2026 that every Gradio Space on its platform now automatically serves an /agents.md endpoint. This endpoint exposes a structured, machine-readable description of what the Space does and how to invoke it, giving AI coding agents a standard way to discover and use a Space's capabilities on their own.

Developers can point agents such as Claude Code, Codex, or Pi directly at a Space's URL, and the agent will read the /agents.md file to understand the Space's interface without any pre-configuration. Hugging Face frames this as eliminating the manual wiring previously required to connect an AI agent to a hosted Gradio application.

The feature arrives alongside a separate April 15 update introducing Kernels repositories on the platform. These provide precompiled, hardware-specific kernel optimizations for PyTorch, with Hugging Face reporting speed-ups of 1.7–2.5× over baseline PyTorch. Kernels are available through the new Kernels Hub.

Why this matters

  • AI coding agents can now autonomously discover and call Gradio Spaces without developers manually writing glue code or API wrappers.
  • The /agents.md endpoint establishes a lightweight open convention that any AI agent or tool could adopt, potentially standardizing how agents interact with web-hosted AI applications.
  • The companion Kernels feature delivers measurable GPU performance gains for PyTorch workloads, which could meaningfully accelerate model training and inference on Hugging Face infrastructure.
