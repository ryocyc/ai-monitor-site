Mistral Ships Voxtral TTS, Mistral Small 4, and Leanstral in March Update

Mistral closed out March with four model releases including a text-to-speech model, a hybrid instruct-reasoning model, and an open-source formal proof agent.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

Mistral released Voxtral TTS (voxtral-tts-2603) on March 23, describing it as a state-of-the-art text-to-speech model supporting zero-shot voice cloning, multilingual output, and real-time streaming. This marks Mistral's entry into the TTS space, adding a voice synthesis capability to a lineup that already covers chat, code generation, transcription, and document processing.

On March 16, the company released Mistral Small 4 (mistral-small-2603), a hybrid model combining instruct, reasoning, and coding tasks in a single multimodal package with a 256,000-token context window. The same day also brought Leanstral (labs-leanstral-2603), which Mistral calls its first open-source code agent purpose-built for Lean 4 formal proof engineering — a niche but growing corner of the AI tooling landscape.

The month's releases were rounded out by Mistral Moderation 2603 on March 12, alongside API-level additions for Custom Guardrails that can now be applied at the agent level, on POST /v1/conversations, or on POST /v1/chat/completions.

Why this matters

  • Voxtral TTS marks Mistral's first direct entry into real-time voice synthesis, a crowded market where zero-shot cloning and streaming are key differentiators.
  • Mistral Small 4's hybrid approach — bundling instruct, reasoning, and coding into one 256k-context model — signals continued competition in the efficient all-purpose model tier.
  • Leanstral's focus on Lean 4 formal proof engineering targets a specialized developer audience working in mathematically rigorous code verification, an area few major AI providers have addressed.
