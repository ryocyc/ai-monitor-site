Replicate Publishes Agent Skills for Claude Code, Codex, and Other Coding Assistants

Replicate has released a collection of markdown-based agent skills that give AI coding tools expert-level knowledge about discovering models, calling APIs, and writing prompts for image and video generation.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

Replicate now publishes agent skills — a set of markdown instruction files that equip coding assistants like Claude Code, OpenCode, and OpenAI Codex with specialist knowledge about working with AI models on the platform. The skills cover model discovery, comparison, and API execution, as well as detailed prompting techniques for both image and video generation models.

The skills follow the open Agent Skills specification and are designed to complement, rather than replace, Replicate's existing MCP server. While MCP provides coding assistants with direct API tools, the new skills give them contextual knowledge — such as which models to choose for a given task, how to structure prompts effectively, and what tradeoffs to consider.

Installation is handled through a single command: npx skills add replicate/skills. This pulls all of Replicate's skills into a project and configures them for the user's coding assistant automatically. The skills are available via the agent skills reference and the GitHub repository.

Why this matters

  • Coding assistants can now interact with Replicate's model catalog more intelligently, picking the right model and writing better prompts without manual lookup.
  • The skills complement Replicate's MCP server, creating a two-layer system where MCP handles tool access and skills handle tool wisdom.
  • The open Agent Skills spec means these capabilities can be adopted by any compatible coding assistant, not just Claude Code.
