AWS Security Agent and DevOps Agent Exit Preview, Now Generally Available

AWS has moved two of its AI-powered frontier agents — Security Agent for on-demand penetration testing and DevOps Agent — from preview into general availability.

AWS Machine Learning Blog 2026-04-23 17:08 UTC Key: aws-machine-learning-blog::identity::aws-machine-learning-blog-artificial-intellige Confidence: weak Mode: claude

Article body

AWS has made two of its autonomous AI agents broadly available: AWS Security Agent, which handles on-demand penetration testing, and AWS DevOps Agent, built to assist with cloud operations at scale. Both tools were introduced under the 'frontier agents' banner at re:Invent and have now exited their preview period. These systems are designed to work independently for extended stretches — running for hours or days without continuous human oversight — and to handle multiple concurrent tasks simultaneously. Early feedback from customers and partners who used the preview version of AWS Security Agent was positive, with testers reporting that the agent compressed the kinds of security assessments that traditionally require significant manual effort.

Why this matters

  • Autonomous agents that can run for hours or days unattended represent a meaningful shift in how security testing and cloud operations can be scoped and executed.
  • AWS Security Agent's move to general availability signals that on-demand, AI-driven penetration testing is now a supported production capability on AWS.
  • The DevOps Agent's broad availability expands the set of AI tools AWS offers for managing and operating cloud infrastructure at scale.
