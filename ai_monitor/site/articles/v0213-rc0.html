Ollama has tagged v0.21.3-rc0 as its newest pre-release, published on 24 April 2026. The standout change in this release is a pull request (#15789) that maps OpenAI's "reasoning_effort" parameter onto Ollama's own "think" configuration, giving developers a direct compatibility shim when migrating or proxying OpenAI API calls through Ollama.

The change is oriented toward OpenAI-compatible endpoints, allowing workloads that rely on the upstream "reasoning_effort" field to pass through Ollama without code modification. As a release candidate, v0.21.3-rc0 is subject to further testing before a stable tag is issued. Users are encouraged to test the release against their specific model and inference configurations and report any regressions in the project's issue tracker.
