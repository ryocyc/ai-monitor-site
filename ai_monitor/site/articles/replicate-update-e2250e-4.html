Replicate launches Agent Skills for AI coding assistants

Replicate now publishes structured markdown skills that give AI coding assistants expert-level knowledge about its platform, models, and APIs.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

Replicate has published a set of agent skills — lightweight markdown instruction files designed to equip AI coding assistants with practical knowledge about working on its platform. Announced April 21, 2026, the skills cover model discovery, model comparison, API execution, and prompting strategies tailored to image and video generation models hosted on Replicate.

The skills follow the open Agent Skills specification and are compatible with tools including Claude Code, OpenAI Codex, and OpenCode. They install via a single command — `npx skills add replicate/skills` — which plumbs all available skills into the coding assistant's environment automatically. Users can consult the agent skills reference or the GitHub repository for more detail.

Replicate frames the new skills as a complement to its existing MCP server. Where the MCP server exposes API tools to a coding assistant, the skills layer in the judgment needed to use those tools well — which model fits a given task, how to phrase prompts, and what tradeoffs to keep in mind.

Why this matters

  • Reduces friction when using Replicate programmatically: coding assistants can reason about model selection and prompting rather than relying on generic defaults.
  • Adopts an open spec, signaling interoperability with a broader ecosystem of agent tooling beyond Replicate's own platform.
  • Positions Replicate alongside platforms like Anthropic and OpenAI in the emerging pattern of pairing tool access (MCP) with contextual knowledge (skills).
