aider v0.86.0 Ships GPT-5, Grok-4, and Gemini Flash Lite Support

The latest stable release of the AI pair-programming tool aider adds first-class support for GPT-5 models, Grok-4, and a lightweight Gemini variant.

GitHub aider Releases 2026-04-22 15:01 UTC Key: github-aider-releases::identity::github-aider-releases-aider-release-activi Confidence: moderate Mode: claude

Article body

aider v0.86.0, released by maintainer paul-gauthier, brings expanded model coverage to the open-source AI coding assistant. The release adds support for all GPT-5 models, Grok-4 via both xai/grok-4 and openrouter/x-ai/grok-4 model names, and gemini/gemini-2.5-flash-lite-preview-06-17, a contribution from Tamir Zahavi-Brunner. A new openrouter/moonshotai/kimi-k2 model was also added, with code from Jack Harrington.

On the UX side, the /clear command now prints a confirmation message, making it clear when chat history has been wiped. The /undo output was tightened to show only the first line of each commit message, improving readability. A longstanding bug where new settings for an existing model failed to override old ones was fixed by Andrew Grigorev, and PostHog analytics were updated to match the latest SDK event-capture API. The litellm dependency was bumped to v1.75.0.

Subsequent version bumps (v0.86.1 through v0.86.3.dev) carry no detailed release notes beyond version metadata, while v0.86.2 appears to be a plain version bump tag. The release notes indicate that Aider itself authored roughly 88% of the code in the v0.86.0 release.

Why this matters

  • GPT-5 model support keeps aider competitive with the latest frontier models from OpenAI.
  • Multi-source Grok-4 support (direct xAI and via OpenRouter) gives users deployment flexibility.
  • The /undo readability fix and /clear confirmation message are small but tangible quality-of-life improvements for daily users.
