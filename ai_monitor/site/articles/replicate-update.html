Replicate Publishes Agent Skills for AI Coding Assistants

Replicate has released a collection of markdown instruction files designed to give coding assistants — including Claude Code, OpenAI Codex, and OpenCode — expert-level knowledge about working with its AI model catalog.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

The new Agent Skills are markdown files that encode Replicate-specific knowledge: how to discover models, compare alternatives, call the API correctly, and write effective prompts for image and video generation. Rather than requiring a developer to learn Replicate's conventions through trial and error, the skills let a coding assistant figure it out directly during a task.

The skills follow the open Agent Skills specification and are installed via a single command — `npx skills add replicate/skills` — which drops the files into a project and auto-configures them for the connected coding assistant. Replicate frames the skills as a counterpart to its existing MCP server: the MCP server hands an assistant API tools, while the skills tell it which tools to pick, how to phrase prompts, and what tradeoffs to weigh.

Why this matters

  • Developers can delegate model selection and prompting decisions to an AI assistant that already knows Replicate's quirks, reducing back-and-forth iteration.
  • The open specification means the skills are portable and community-editable, not locked to a single vendor's ecosystem.
  • By pairing skills with the MCP server, Replicate offers a more complete developer experience — tools plus institutional knowledge — that could lower the barrier for teams building AI-powered workflows.
