Ollama v0.21.2-rc1 Brings Bundled OpenClaw and Structured Output Docs

The Ollama team has shipped a release candidate that integrates OpenClaw directly for native web search capabilities, alongside new documentation covering structured outputs and Ollama Cloud.

Article body

Ollama v0.21.2-rc1 was released on April 23, 2026, tagging two distinct changes. First, contributor @madflow added documentation for structured outputs and Ollama Cloud in pull request #15733. Second, @ParthSareen merged a change that launches a bundled OpenClaw to enable native web search directly through Ollama, according to PR #15757.

The bundled OpenClaw integration marks a practical shift: rather than relying on external search tooling, Ollama ships the capability directly. This should make web-augmented generation simpler to configure in local and self-hosted setups, without requiring additional package management.

The release also marks @madflow's first contribution to the project. The full changelog covers changes since v0.21.1.
