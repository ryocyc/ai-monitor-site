Replicate Publishes Agent Skills to Sharpen AI Coding Assistants

Replicate has started publishing Agent Skills — markdown instruction files that give coding assistants expert knowledge for working with its models, published as a companion to its MCP server.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

Replicate is now distributing a collection of Agent Skills: markdown files that encode expert guidance for coding assistants navigating its platform. Announced on April 21, 2026, the skills cover model discovery, model comparison, API execution, and prompting techniques specifically for image and video generation models. They are installed with a single command, `npx skills add replicate/skills`, which drops the files into a project and configures the coding assistant automatically.

The skills follow the open Agent Skills specification and are explicitly tested against Claude Code, OpenCode, and OpenAI Codex. They are positioned as a complement to Replicate's existing MCP server. Where the MCP server exposes raw API tools to a coding assistant, the new skills layer on context about how to use those tools effectively — which models suit which tasks, how to frame prompts, and what tradeoffs come into play.

The distinction is deliberately framed as tools versus knowledge. MCP gives a coding assistant what it can do; Agent Skills teach it how to do it well on Replicate specifically.

Why this matters

  • Coding assistants working with Replicate's model catalog often lack platform-specific guidance, leading to suboptimal model choices or generic prompting — skills files address that gap directly.
  • The open Agent Skills spec means the format is not locked to Replicate's ecosystem; compatible tools beyond Replicate can in principle consume the same files.
  • Installation via a standard `npx` command makes the skills immediately accessible to developers already working in Node.js environments, with no manual configuration required.
