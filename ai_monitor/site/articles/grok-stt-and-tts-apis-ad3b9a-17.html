xAI Launches Grok Speech-to-Text and Text-to-Speech APIs

xAI has released two standalone audio APIs giving developers direct access to Grok-powered speech recognition and synthesis.

xAI Blog 2026-04-22 23:49 UTC Key: xai-blog::identity::grok-stt-and-tts-apis Confidence: strong Mode: claude

Article body

xAI announced on April 17, 2026 the launch of Grok Speech to Text and Grok Text to Speech, two independent API endpoints built on the same infrastructure that already powers Grok Voice in Tesla vehicles and Starlink customer support. The STT API handles both batch transcription of large audio files and real-time streaming via WebSocket, offering features like word-level timestamps, speaker diarization, and multichannel audio separation. The TTS API converts long-form text to speech using a REST endpoint or a streaming WebSocket connection, and supports inline speech tags to control prosody and emotion, including commands like [laugh], [sigh], and [whisper].

The company is positioning Grok STT as a direct competitor to ElevenLabs, Deepgram, and AssemblyAI, publishing benchmark figures showing an overall word error rate of 6.9% versus 9.0%, 11.0%, and 12.9% respectively across phone calls, video, meetings, and telephony domains. The API also performs Inverse Text Normalization to intelligently format transcribed output, converting raw spoken numbers, dates, and currencies into structured text rather than leaving them as verbatim dictation.

Pricing is set at $0.10 per hour for batch STT and $0.20 per hour for streaming STT, while TTS is priced at $4.20 per one million characters. Both APIs are documented for use cases spanning voice agents, real-time transcription tools, accessibility applications, podcasts, and interactive audio experiences, and support more than 25 languages with seamless switching between them.

Why this matters

  • xAI is competing directly with established speech API providers using benchmark data that positions Grok STT ahead on word error rate across all tested domains.
  • The APIs share infrastructure with Tesla and Starlink deployments, signaling real-world use at scale rather than lab-only performance.
  • Inline speech control tags like [laugh] and [whisper] give developers a rare fine-grained tool to shape TTS delivery without complex markup.
