Replicate Launches Agent Skills: Expert Instructions for AI Coding Assistants

Replicate has begun publishing "agent skills"—markdown instruction files that give AI coding assistants structured knowledge about working with its models and API.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: moderate Mode: claude

Article body

Replicate's new agent skills are markdown files that encode expert-level guidance for coding assistants such as Claude Code, OpenAI Codex, and OpenCode. Rather than fumbling through documentation, a compatible assistant can read these files to understand how to discover the right model for a task, execute API calls correctly, and write effective prompts for image and video generation. The skills are installed via a single command: `npx skills add replicate/skills`, which pulls the full set into a project and auto-configures the assistant.

The skills follow the open Agent Skills specification, meaning they are portable across any toolchain that adopts the same format. Replicate positions them as a complement to its existing MCP server. Where the MCP server exposes raw API endpoints as callable tools, the skills layer in the judgment call of which model to pick, how to phrase a prompt, and what tradeoffs to weigh—knowledge that raw tool access alone cannot convey.

Skills are available immediately for Replicate users running a compatible coding assistant. The changelog notes a GitHub repository and an agent skills reference for further detail, though neither is expanded upon in the entry itself.

Why this matters

  • Closes a gap between having API access and knowing how to use it well inside an AI coding workflow.
  • Adoption of an open spec means skills could become reusable across multiple platforms that commit to the same format.
  • The MCP + Skills division of labor suggests a broader pattern for how AI tooling will separate "can do" (tools) from "knows how" (skills).
