Mistral Drops Four March Releases Spanning TTS, Reasoning, and Formal Proofs

Mistral shipped four model releases in March 2026, covering text-to-speech, a hybrid instruct/reasoning model, a Lean 4 code agent, and an updated moderation classifier.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: moderate Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a text-to-speech model supporting zero-shot voice cloning, multilingual output, and real-time streaming. The release marks Mistral's entry into the audio generation space, offering what the changelog calls a "state-of-the-art" TTS model. Separately, on March 16 the company released Mistral Small 4 (mistral-small-2603), a multimodal model combining instruct, reasoning, and coding capabilities in a single system with a 256k-token context window, positioning it as a hybrid alternative to purpose-built reasoning models.

The same date brought Leanstral (labs-leanstral-2603), Mistral's first open-source code agent built specifically for Lean 4 formal proof engineering. Leanstral represents a niche but growing corner of the AI tooling landscape, targeting mathematicians, verification engineers, and researchers working in formal methods. The March 12 release of Mistral Moderation 2603 (mistral-moderation-2603) rounds out the month's model catalog, updating the company's content moderation classifier.

Alongside the model releases, Mistral extended its Custom Guardrails feature across three API surfaces: the `guardrails` parameter on Agents, the `guardrails` field on POST /v1/conversations, and the equivalent field on POST /v1/chat/completions. This allows developers to apply safety policies per-agent or per-request without routing logic, giving more granular control than the previous global configuration approach.

Why this matters

  • Voxtral TTS marks Mistral's first entry into voice generation, expanding beyond its text-only and audio-transcription models.
  • Mistral Small 4's 256k context and hybrid instruct-reasoning-coder design signals a push toward a single model that handles diverse developer workloads.
  • Leanstral targets a specialized formal-verification audience, an underserved area in the current open-source agent ecosystem.
