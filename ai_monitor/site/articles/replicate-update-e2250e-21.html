Replicate Publishes Agent Skills for AI Coding Assistants

Replicate now offers Agent Skills — markdown instruction files that give coding assistants expert-level knowledge about working with its models and API.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

Replicate has published a set of Agent Skills: markdown files that encode expert knowledge about interacting with its platform. Coding assistants including Claude Code, OpenAI Codex, and OpenCode can read these files to learn how to discover models, compare them, execute predictions via the API, and write effective prompts for image and video generation tasks.

The skills follow the open Agent Skills specification and install with a single command: `npx skills add replicate/skills`. Once installed, they integrate with the assistant's workflow automatically. Replicate frames skills and its existing MCP server as complementary — the MCP server provides API tools, while skills provide the context for using those tools well, including guidance on model tradeoffs and prompting strategies.

The release is dated April 21, 2026 on Replicate's changelog, alongside a handful of smaller recent additions: a fallback model option for Nano Banana Pro (falling back to ByteDance Seedream 5.0 lite under rate limits), auto-discovery metadata for MCP clients via a well-known endpoint, and a `source=web` filter on the predictions API.

Why this matters

  • Agent Skills let coding assistants use Replicate more effectively without manual lookup, potentially lowering the barrier to programmatic model invocation.
  • The open Agent Skills spec means skills written for Replicate may be reusable across other tools and platforms that adopt the same format.
  • By pairing skills with its MCP server, Replicate positions itself as an AI-assistant-friendly inference platform rather than a pure API vendor.
