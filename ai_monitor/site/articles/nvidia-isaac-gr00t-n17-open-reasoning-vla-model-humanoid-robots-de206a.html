Article body

The Isaac GR00T N1.7 model is now available on Hugging Face as an open-source release from NVIDIA, representing an expansion of the company's open-weight robotics model family. The model is described as a Vision-Language-Action (VLA) system, combining visual perception, natural language understanding, and robot control in a single framework aimed at humanoid platforms.

As an open-weight model, researchers and developers can access the model weights directly through Hugging Face and adapt the system for their own robotics projects. The VLA architecture is designed to let the robot interpret visual input, reason about language-based instructions, and generate corresponding physical actions.
