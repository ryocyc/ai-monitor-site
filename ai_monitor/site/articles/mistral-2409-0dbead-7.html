Mistral March 2026 Changelog: Voxtral TTS, Mistral Small 4, Leanstral, and More

Mistral shipped four new or updated models in March 2026, spanning text-to-speech, general-purpose instruction, formal proof engineering, and content moderation.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a text-to-speech model featuring zero-shot voice cloning, multilingual support, and real-time streaming. The addition marks Mistral's entry into the speech synthesis space, rounding out a portfolio that has previously focused on text and code models.

March 16 brought Mistral Small 4 (mistral-small-2603), a hybrid model that unifies instruct, reasoning, and coding capabilities in a single multimodal package with a 256,000-token context window. That same date also saw the release of Leanstral (labs-leanstral-2603), described as Mistral's first open-source code agent purpose-built for Lean 4 formal proof engineering — a niche but technically demanding corner of the software verification world.

Earlier in the month, on March 12, Mistral shipped Mistral Moderation 2603 (mistral-moderation-2603). The API layer also received an update: Custom Guardrails can now be configured directly on an Agent via a dedicated parameter, or passed per-request on both the POST /v1/conversations and POST /v1/chat/completions endpoints.

Why this matters

  • Voxtral TTS brings Mistral into direct competition with established speech synthesis providers, and its zero-shot cloning capability suggests a focus on voice personalization use cases.
  • Mistral Small 4's 256k context window is unusually large for a hybrid instruct/reasoning model, potentially making it suitable for long-document tasks that previously required separate retrieval steps.
  • Leanstral targets a specialised but growing community around formal methods and interactive proof assistants, signaling Mistral's intent to serve not just mainstream LLM consumers but also technical research niches.
