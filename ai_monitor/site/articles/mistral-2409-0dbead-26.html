Mistral March Releases: Voxtral TTS, Mistral Small 4, Leanstral, Moderation 2603

Mistral's March 2025 changelog details four model releases including a new text-to-speech system with voice cloning, a hybrid instruct-reasoning model, and an open-source formal proof agent.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

Mistral released Voxtral TTS (voxtral-tts-2603) on March 23, a text-to-speech model described as state-of-the-art with zero-shot voice cloning, multilingual support, and real-time streaming. This builds on earlier Voxtral audio work that began with small chat models and transcription tools in July 2024.

On March 16, the company shipped Mistral Small 4 (mistral-small-2603), positioning it as a hybrid model that unifies instruct, reasoning, and coding tasks in a single multimodal package with a 256,000-token context window. The same date also saw the release of Leanstral (labs-leanstral-2603), Mistral's first open-source code agent purpose-built for Lean 4 formal proof engineering.

March 12 brought Mistral Moderation 2603 (mistral-moderation-2603), with the changelog noting that Custom Guardrails support was simultaneously added for Agents and Conversations, allowing guardrails to be configured at the agent level and passed per-request on both the POST /v1/conversations and POST /v1/chat/completions endpoints.

Why this matters

  • Voxtral TTS is Mistral's most capable TTS entry, adding zero-shot voice cloning and real-time streaming to a product line that previously offered only smaller chat and transcription variants.
  • Mistral Small 4's 256k context window is notably large for a hybrid instruct-reasoning model, potentially simplifying pipelines that previously required separate models for different tasks.
  • Leanstral is Mistral's first open-source code agent for formal proof engineering, marking a new direction into certified software correctness using the Lean 4 proof language.
