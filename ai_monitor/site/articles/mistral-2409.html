Mistral March Releases: Voxtral TTS, Mistral Small 4 with 256k Context, Leanstral, Moderation 2603

Mistral shipped a wave of releases across March 2025, spanning a text-to-speech model, a unified hybrid model, an open-source formal proof agent, and a moderation model, alongside expanded Guardrails API controls.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: moderate Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a text-to-speech model featuring zero-shot voice cloning, multilingual support, and real-time streaming. The model was flagged as a new release in the model category.

March 16 brought two notable releases. Mistral Small 4 (mistral-small-2603) was introduced as a hybrid model that unifies instruct, reasoning, and coding capabilities in a single multimodal package, offering a 256k context window. On the same day, Mistral released Leanstral (labs-leanstral-2603), its first open-source code agent purpose-built for Lean 4 formal proof engineering.

On March 12, Mistral released Mistral Moderation 2603 (mistral-moderation-2603) as a new model. Separately, Mistral added Custom Guardrails support for Agents and Conversations, with guardrails configurable directly on an Agent, or passed per-request via the guardrails field on POST /v1/conversations and POST /v1/chat/completions endpoints.

Why this matters

  • Voxtral TTS marks Mistral's entry into real-time speech synthesis with voice cloning, expanding beyond its language model catalog.
  • Mistral Small 4's 256k context window positions it against large-context competitors for long-document tasks.
  • Leanstral's open-source formal proof agent bridges Mistral into the academic and verification tooling space.
