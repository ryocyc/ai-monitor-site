Mistral March Releases: Voxtral TTS, Mistral Small 4, Leanstral, Moderation 2603

Mistral shipped four distinct releases in March 2025 spanning speech synthesis, a hybrid instruct-reasoning model, a formal proof agent, and a moderation update.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

Mistral released Voxtral TTS (voxtral-tts-2603) on March 23, calling it a state-of-the-art text-to-speech model. The release description highlights zero-shot voice cloning, multilingual support, and real-time streaming capabilities. This marks Mistral's entry into the speech synthesis space, following earlier audio releases in 2024.

On March 16, Mistral released Mistral Small 4 (mistral-small-2603), described as a hybrid model that unifies instruct, reasoning, and coding in a single multimodal model with a 256k context window. The changelog flags this as a model release, though no benchmark numbers or comparative details are provided on this page.

The same day also brought Leanstral (labs-leanstral-2603), Mistral's first open-source code agent designed for Lean 4 formal proof engineering. Mistral Moderation 2603 (mistral-moderation-2603) followed on March 12 as another model release. Additionally, the changelog notes that Custom Guardrails support was added to the Agents and Conversations API, with guardrail parameters now configurable at the agent level, per-request on POST /v1/conversations, and per-request on POST /v1/chat/completions.

Why this matters

  • Voxtral TTS gives developers a TTS option with voice cloning from Mistral's ecosystem, competing with services from OpenAI and others.
  • Mistral Small 4's 256k context window and hybrid instruct-reasoning-coder design suggests a response to the growing market for long-context coding models.
  • Leanstral brings formal proof engineering into Mistral's open-source lineup, appealing to mathematics and verification-focused developers.
