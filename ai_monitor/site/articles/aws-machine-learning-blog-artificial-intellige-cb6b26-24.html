AWS Security Agent and DevOps Agent Exit Preview, Now Generally Available

AWS has moved its Security Agent and DevOps Agent out of preview, making both frontier agents available to customers for autonomous security testing and cloud operations.

Article body

AWS Security Agent and AWS DevOps Agent are now generally available, representing a new class of AI capabilities the company calls frontier agents. These autonomous systems can work independently for hours or days without constant human oversight, handling concurrent tasks at scale. The Security Agent is designed for on-demand penetration testing, while the DevOps Agent handles cloud operations tasks.
