Mistral Ships Voxtral TTS, Mistral Small 4, and Leanstral in March

Mistral's March updates brought a text-to-speech model with zero-shot voice cloning, a hybrid instruct-reasoning model with a 256k context window, and an open-source agent purpose-built for formal proof engineering.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a text-to-speech model supporting zero-shot voice cloning, multilingual output, and real-time streaming. The release marks Mistral's first entry into the voice synthesis space on its platform.

March 16 saw the arrival of Mistral Small 4 (mistral-small-2603), a single multimodal model combining instruct, reasoning, and coding capabilities within a 256k-token context window. The same day, Mistral published Leanstral (labs-leanstral-2603), an open-source code agent focused on Lean 4 formal proof engineering, making it Mistral's first dedicated formal verification tool.

On March 12, Mistral Moderation 2603 (mistral-moderation-2603) shipped alongside the addition of Custom Guardrails support for Agents and Conversations. The new guardrails parameter can now be applied at the agent level, or per-request on both POST /v1/conversations and POST /v1/chat/completions.

Why this matters

  • Voxtral TTS with real-time streaming positions Mistral as a direct competitor in the speech synthesis market, an area previously dominated by a small number of specialized providers.
  • Mistral Small 4's hybrid instruct-reasoning design in a single model, paired with a 256k context window, aims to reduce the need for chaining separate models for complex tasks.
  • Leanstral signals Mistral's move into formal proof engineering tooling, an academic-adjacent niche where open-source tooling has historically been sparse.
