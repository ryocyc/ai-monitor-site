Hugging Face Adds Machine-Readable /agents Endpoint to All Gradio Spaces

Every Gradio Space now exposes a standardized /agents.md endpoint, letting AI coding agents like Claude Code, Codex, and Pi discover and invoke Space capabilities without any manual configuration.

Hugging Face Changelog <span data-utc="2026-04-26T10:29:04+00:00">2026-04-26 10:29 UTC</span> Key: evt-b2e6e7c8e4d630c20553 Confidence: moderate Mode: claude

Article body

Hugging Face has updated its platform so that every Gradio Space automatically serves an /agents.md endpoint — a machine-readable API description that AI coding agents can parse and act on directly. The feature was listed on the Hugging Face changelog on April 17, 2026, and carries 139 upvotes from the community.

The /agents.md file functions as a self-documenting interface: when a coding agent encounters a Space, it reads the endpoint to understand what the Space does and how to call it, removing the need for developers to write custom wrappers or integration code. The changelog specifically cites Claude Code, Codex, and Pi as examples of agents that can consume this format, though the endpoint is designed to work with any compatible agent.

For developers building agent-based workflows, the change reduces friction at the integration layer. Rather than requiring Spaces authors to provide documentation or users to configure agent prompts manually, the /agents.md endpoint allows agents to self-discover Space capabilities at runtime. This mirrors the broader industry shift toward standardized tool-description formats that let agents interoperate across platforms without bespoke glue code.

Why this matters

  • Eliminates manual integration work: agents can interact with any Gradio Space out of the box, without per-Space configuration.
  • Lowers the bar for agent-aware Spaces: Space authors benefit automatically with no additional effort on their part.
  • Pushes toward a common agent tooling interface: standardized machine-readable tool descriptions reduce fragmentation across the agent ecosystem.
