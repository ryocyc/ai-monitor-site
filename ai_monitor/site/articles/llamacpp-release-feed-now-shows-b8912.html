Article body

The ggml-org/llama.cpp project shipped release b8912 on 24 April 2026, authored by a GitHub Actions workflow and signed with GitHub's verified GPG key. The single commit — fa0b8a7 — addresses the third item from issue #20429, removing local struct variables that are now redundant because their values are already stored in defaults.sampling.
