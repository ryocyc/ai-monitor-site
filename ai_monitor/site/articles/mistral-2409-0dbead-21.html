Mistral March Releases: Voxtral TTS, Mistral Small 4, Leanstral, and Moderation 2603

Mistral shipped four notable releases in March 2025, spanning a new text-to-speech model, a hybrid reasoning-instruct model, an open-source formal proof agent, and an updated moderation model.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a text-to-speech model featuring zero-shot voice cloning, multilingual support, and real-time streaming capabilities. The release positions Voxtral as Mistral's first dedicated audio output model.

March 16 brought two releases in parallel. Mistral Small 4 (mistral-small-2603) is a hybrid model combining instruct, reasoning, and coding tasks in a single multimodal package, with a 256,000-token context window. Also on that date, Mistral released Leanstral (labs-leanstral-2603), described as the company's first open-source code agent purpose-built for Lean 4 formal proof engineering.

A third March release, Mistral Moderation 2603 (mistral-moderation-2603), landed on March 12. The same period also expanded Custom Guardrails support: guardrails can now be configured directly on an Agent, and passed per-request via both POST /v1/conversations and POST /v1/chat/completions.

Why this matters

  • Voxtral TTS adds a new audio modality to Mistral's offering, directly competing with established TTS providers in a model release rather than a separate product.
  • Mistral Small 4's hybrid approach merges instruct, reasoning, and coding into one model, with a 256k context window that is large even by frontier-model standards.
  • Leanstral marks Mistral's first explicit entry into formal proof engineering tooling, an area typically dominated by specialized academic projects.
