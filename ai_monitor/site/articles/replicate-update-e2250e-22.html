Replicate Publishes Agent Skills for AI Coding Assistants

Replicate has released a set of markdown instruction files called agent skills that give coding assistants expert-level knowledge about discovering, comparing, and working with AI models on the platform.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

Replicate now publishes agent skills, a collection of markdown instruction files designed to give coding assistants such as Claude Code, OpenCode, and OpenAI Codex expert knowledge about working with AI models on the platform. The skills cover model discovery, comparison, and API execution, as well as detailed prompting techniques for image generation and video generation models. They follow the open Agent Skills specification and can be installed in one command: `npx skills add replicate/skills`.

Once installed, the skills configure themselves for the coding assistant automatically. Replicate emphasises that these agent skills are complementary to its existing MCP server, which provides raw API tools. Where MCP hands a coding assistant the API calls, agent skills supply the contextual knowledge to use those calls well—covering which models to pick, how to structure prompts, and what tradeoffs to consider.

The changelog notes that further details, including a reference guide and a link to the GitHub repository, are available for developers who want to explore the full scope of what the skills cover.

Why this matters

  • Coding assistants gain structured, model-specific knowledge without manual prompt engineering or research.
  • The one-step install (`npx skills add replicate/skills`) lowers the barrier for integrating Replicate into AI-augmented development workflows.
  • Agent skills and the MCP server are complementary tools, letting assistants both call the API and reason about how to use it effectively.
