Mistral's March Releases: Voxtral TTS, Mistral Small 4 (256k Context), Leanstral, and Moderation 2603

Mistral shipped a cluster of notable releases in mid-to-late March spanning speech synthesis, a hybrid instruct-reasoning model, an open-source formal-proof code agent, and an updated moderation model.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a state-of-the-art text-to-speech model featuring zero-shot voice cloning, multilingual support, and real-time streaming capabilities. This marks Mistral's entry into the speech synthesis space and complements its existing audio transcribe and chat offerings under the Voxtral family.

On March 16, the company released Mistral Small 4 (mistral-small-2603), a hybrid model that unifies instruct, reasoning, and coding tasks in a single multimodal package with a 256,000-token context window. The same day also saw the debut of Leanstral (labs-leanstral-2603), described as Mistral's first open-source code agent purpose-built for Lean 4 formal proof engineering.

On March 12, Mistral shipped Mistral Moderation 2603 (mistral-moderation-2603) alongside the addition of Custom Guardrails support for Agents and Conversations, allowing guardrails to be configured at the agent level and passed per-request via the POST /v1/conversations and POST /v1/chat/completions endpoints.

Why this matters

  • Voxtral TTS expands Mistral's audio portfolio into synthesis, competing with established TTS providers using a zero-shot cloning approach.
  • Mistral Small 4's 256k context window is notably large for a hybrid instruct-reasoning model, enabling longer document and code review workloads.
  • Leanstral is Mistral's first open-source code agent aimed at formal verification, a niche but growing area in AI-assisted software engineering.
