Replicate Publishes Agent Skills to Guide AI Coding Tools on Model Discovery and Generation

Replicate has released a set of markdown instruction files called Agent Skills, designed to give AI coding assistants expert-level knowledge about working with its platform.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: moderate Mode: claude

Article body

Replicate now publishes Agent Skills — a collection of markdown instruction files that equip AI coding assistants with structured knowledge about using models on the platform. The skills cover model discovery, model comparison, and API execution, along with detailed prompting techniques for image and video generation models. They follow the open Agent Skills specification and are confirmed to work with Claude Code, OpenCode, and OpenAI Codex.

The skills install with a single command: `npx skills add replicate/skills`, which pulls all of Replicate's skills into a project and configures them for the coding assistant automatically. Replicate frames the release as complementary to its existing MCP server. Where the MCP server provides API tools to a coding assistant, Agent Skills provide the contextual knowledge to use those tools well — including which models to pick, how to frame prompts, and what tradeoffs to consider.

The changelog also notes two other recent additions: Nano Banana Pro now optionally falls back to ByteDance Seedream 5.0 lite when Google's API hits capacity limits, and Replicate's MCP server can now be discovered automatically through the official MCP Registry via a `/.well-known/mcp/server.json` endpoint.

Why this matters

  • Agent Skills formalizes how third-party coding assistants should work with Replicate, going beyond raw API access to encode best practices for model selection and prompting.
  • The open Agent Skills spec means skills published by Replicate are potentially reusable across any compatible assistant, not just Replicate's own tools.
  • For developers using Claude Code, OpenAI Codex, or OpenCode, the installation path (`npx skills add replicate/skills`) makes it straightforward to add Replicate-specific expertise to an existing workflow.
