Ollama, the open-source runtime for running large language models locally, has published a release candidate tagged v0.22.1-rc1 to its GitHub releases page. The single changelog entry in this release updates the Gemma4 renderer, touching PR #15886. The release was pushed on 30 April 2026. Gemma4 is Google's family of open-weight transformer models, and the renderer referenced here is the component within Ollama responsible for token stream handling and output formatting when running Gemma4-based prompts. The change signals continued compatibility work between Ollama's inference layer and Google's latest model weights. For developers running Ollama as a local model server, the release candidate offers an early path to test the Gemma4 renderer update before it lands in a stable tag.
