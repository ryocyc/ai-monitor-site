Article body

On April 23, 2026, the n8n project tagged version 2.17.6 as the latest stable release. The single change in this release is a bug fix to core user search functionality used within ProjectSettings, linked to pull request #28949 and committed as d388990. The fix resolves an issue where user search behavior was not functioning correctly in that particular settings context.
