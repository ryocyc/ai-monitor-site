n8n 2.18.4 Rolls Out AI Wallet Fixes and Moves Computer Use Toggle to Chat Sidebar

The open-source workflow automation platform n8n has released version 2.18.4, fixing a license-related credit prompt conflict and repositioning its computer use control for AI agent workflows.

Article body

n8n, the open-source workflow automation platform, shipped version 2.18.4 on April 27, 2026. The release includes two targeted bug fixes and one new feature focused on improving the AI agent experience within the platform's workflow editor.

The most significant change resolves a conflict in the license layer: the "Get 100 free OpenAI API credits" prompt is now suppressed when the aiGateway flag is enabled in the license configuration. A second fix corrects a stale wallet balance display by triggering a refresh whenever a user manually refreshes usage data. On the feature side, the computer use toggle — a control for AI agents that operate a remote desktop or environment — has been moved out of its previous location and into the chat sidebar, centralizing it within the conversational agent UI.

For teams building AI-powered automations, these changes tighten the licensing and billing experience. The aiGateway credit suppression matters most to users operating under enterprise or self-hosted license arrangements, while the relocated computer use toggle streamlines the interaction model for agents that rely on desktop-style task execution. All three changes are available immediately by upgrading to n8n@2.18.4.

Why this matters

  • The aiGateway credit prompt fix prevents confusing or misleading billing behavior for users running n8n under managed AI gateway licenses, where free OpenAI credits would not be applicable.
  • The computer use toggle relocation signals continued investment in AI agent capabilities within n8n, giving builders a more accessible control point for agent-driven workflows.
  • Wallet balance refresh consistency helps MLOps and automation teams monitor actual API spend more accurately within the n8n editor, reducing surprises on monthly billing cycles.
