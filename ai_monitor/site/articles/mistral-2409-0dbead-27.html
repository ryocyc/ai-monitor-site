Mistral March Releases: Voxtral TTS, Mistral Small 4, Leanstral, and Moderation 2603

Mistral released a cluster of notable models and API updates in March 2026, including its first text-to-speech model, a new hybrid instruct/reasoning/coding model, an open-source Lean 4 proof agent, and an updated moderation model.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a text-to-speech model described as state-of-the-art, featuring zero-shot voice cloning, multilingual support, and real-time streaming capabilities. This marks Mistral's entry into the audio generation space, complementing its existing Voxtral family of audio models.

March 16 saw two significant model releases. Mistral Small 4 (mistral-small-2603) is a hybrid model that unifies instruct, reasoning, and coding abilities in a single multimodal architecture with a 256k context window. Separately, Leanstral (labs-leanstral-2603) was released as Mistral's first open-source code agent purpose-built for Lean 4 formal proof engineering—a niche but technically demanding domain.

Mistral also shipped Mistral Moderation 2603 on March 12 and expanded Guardrails functionality across its Agents API, allowing configuration at the agent level via the `guardrails` parameter, as well as per-request via the `guardrails` field on both the `/v1/conversations` and `/v1/chat/completions` endpoints.

Why this matters

  • Voxtral TTS is Mistral's first text-to-speech model, signaling expansion beyond text and image into audio generation.
  • Mistral Small 4's 256k context window and hybrid instruct/reasoning/coding design targets developers who need a single capable model for varied tasks.
  • Leanstral's open-source release provides the research community with a dedicated tool for formal mathematics using the Lean 4 proof language.
