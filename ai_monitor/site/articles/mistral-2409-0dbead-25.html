Mistral March 2026 Releases: Voxtral TTS, Mistral Small 4, Leanstral, and Moderation 2603

Mistral shipped four distinct releases in March 2026, spanning a new text-to-speech model, a hybrid instruct-reasoning-coding model, an open-source formal proof agent, and a refreshed moderation model.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: strong Mode: claude

Article body

On March 23, Mistral released Voxtral TTS (voxtral-tts-2603), a text-to-speech model featuring zero-shot voice cloning, multilingual support, and real-time streaming. The release marks Mistral's entry into the audio generation space, offering what the company describes as state-of-the-art voice synthesis capabilities.

March 16 brought two simultaneous releases. The first was Mistral Small 4 (mistral-small-2603), described as a hybrid model that unifies instruct, reasoning, and coding in a single multimodal package with a 256k context window. The second was Leanstral (labs-leanstral-2603), Mistral's first open-source code agent purpose-built for Lean 4 formal proof engineering, positioning the company in the emerging category of formal verification tooling.

On March 12, Mistral released Mistral Moderation 2603 (mistral-moderation-2603), refreshing its content moderation model. The same period also saw Custom Guardrails added as a configurable parameter on Agents and made available per-request on both the POST /v1/conversations and POST /v1/chat/completions API endpoints.

Why this matters

  • Voxtral TTS is Mistral's first dedicated text-to-speech offering, bringing zero-shot voice cloning and multilingual support into the company's model portfolio alongside its existing text and audio transcription models.
  • Mistral Small 4's hybrid design combining instruct, reasoning, and coding in one 256k-context model signals a push toward generalist capabilities in a compact package.
  • Leanstral marks Mistral's first open-source code agent targeting formal proof engineering in Lean 4, an area with growing interest from academic and safety-critical software communities.
