Replicate Publishes Agent Skills for AI Coding Assistants

Replicate now offers agent skills — markdown instruction files that give coding assistants expert knowledge for working with its models and API.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

Replicate has published a collection of agent skills: markdown files that equip AI coding assistants with specialist knowledge about the Replicate platform. According to the changelog, the skills cover model discovery, model comparison, API execution, and prompting techniques tailored to image generation and video generation models.

The skills follow the open Agent Skills specification and are designed to work with Claude Code, OpenCode, OpenAI Codex, and other compatible tools. They install via a single npm command and automatically configure themselves for the target coding assistant. Replicate frames skills as complementary to its existing MCP server — while MCP provides API tools, skills supply the contextual knowledge needed to use those tools well, including guidance on model selection and prompt design.

The changelog also notes a fallback model option for Nano Banana Pro (effective March 2, 2026), allowing it to switch to ByteDance Seedream 5.0 lite when Google's API is at capacity, and MCP server auto-discovery via the official MCP Registry (effective February 10, 2026).

Why this matters

  • The Agent Skills spec is an open standard, meaning Replicate's move could encourage broader tooling interoperability across AI model providers.
  • Coding assistants equipped with these skills can make better-informed decisions about which models to call and how to prompt them, reducing trial and error for developers.
  • By pairing skills with the MCP server, Replicate offers a two-layer integration: tools on one side, domain knowledge on the other.
