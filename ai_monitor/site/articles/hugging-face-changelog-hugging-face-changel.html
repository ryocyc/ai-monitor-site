Gradio Spaces Auto-Serve /agents.md Endpoints for AI Coding Agents

Every Gradio Space on Hugging Face now automatically serves an /agents.md endpoint, allowing AI coding agents to read and use the Space without any manual configuration.

Hugging Face Changelog 2026-04-23 10:43 UTC Key: hugging-face-changelog::identity::hugging-face-changelog-hugging-face-changel Confidence: strong Mode: claude

Article body

Hugging Face has quietly added a capability to every Gradio Space that changes how AI coding agents interact with hosted demos. Every Space now automatically serves an /agents.md endpoint — a machine-readable API description that agents can parse and call directly, no setup or documentation hunting required. Developers can simply point their agent at the endpoint and the agent figures out how to use the Space autonomously. The change is designed to be frictionless on the Space owner's side as well, requiring no additional configuration. Examples cited by Hugging Face include Claude Code, Codex, and Pi, covering the major commercial and research AI coding agents currently in use.

Why this matters

  • Eliminates manual API discovery for AI agents — they can read /agents.md and know exactly how to interact with a Space without human intervention.
  • No extra configuration required from Space creators — the endpoint is served automatically for every Gradio Space on the platform.
  • Enables agents to use third-party Gradio Spaces in real time, opening up the Hub as a live toolkit for AI-assisted development workflows.
