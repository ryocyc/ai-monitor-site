Gradio Spaces Now Auto-Serve /agents.md Endpoints for AI Coding Agents

Every Gradio Space on Hugging Face now automatically exposes an /agents.md endpoint, letting AI coding agents like Claude Code, Codex, and Pi interact with the Space without any manual configuration.

Hugging Face Changelog 2026-04-23 10:43 UTC Key: hugging-face-changelog::identity::hugging-face-changelog-hugging-face-changel Confidence: moderate Mode: claude

Article body

Hugging Face has quietly enabled a new feature that makes Gradio Spaces far easier for AI agents to consume programmatically. Every Space now automatically serves an /agents.md endpoint — a machine-readable API description that AI coding agents can parse and call directly, without requiring human setup or custom integrations. The change applies globally across all existing and new Gradio Spaces.

Agents such as Claude Code, Codex, and Pi can now point themselves at a Space's /agents.md endpoint and automatically understand how to use the Space's interface. The system handles the discovery and invocation layer, eliminating the need for developers to write custom wrappers or documentation parsers to bridge agents with Spaces.

The feature is documented in Hugging Face's official documentation. It has gathered modest community interest, with the changelog entry collecting around 100 upvotes and 95 net positive votes as of mid-April 2026. The changelog entry is part of a broader set of recent updates, including a new Kernels Hub for GPU-optimised precompiled kernels and an Agent Traces upload feature for logging agent sessions to Hugging Face Datasets.

Why this matters

  • Removes a key friction point: AI coding agents can now discover and interact with Gradio Spaces without bespoke setup or wrapper code.
  • Standardises how agents understand Spaces, potentially lowering the barrier to building agent-powered applications on the Hugging Face platform.
  • Complements other recent agent-focused updates — Agent Traces upload and ZeroGPU overquota credits — suggesting a coordinated push to make the Hub a first-class agent environment.
