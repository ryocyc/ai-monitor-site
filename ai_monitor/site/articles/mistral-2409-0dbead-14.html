Mistral March Releases: Voxtral TTS, Mistral Small 4, Leanstral, and Moderation 2603

Mistral shipped four notable releases in March, including a new text-to-speech model, a hybrid instruct/reasoning model with a 256k context window, and its first open-source formal proof agent.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: moderate Mode: claude

Article body

On March 23, Mistral released Voxtral TTS, a text-to-speech model with zero-shot voice cloning, multilingual support, and real-time streaming capabilities. The model is identified as voxtral-tts-2603 and appears to be the company's first dedicated TTS offering.

Mistral Small 4 arrived on March 16, described as a hybrid model that unifies instruct, reasoning, and coding tasks in a single multimodal package with a 256k context window. The model is tagged as mistral-small-2603.

The same day, Mistral released Leanstral (labs-leanstral-2603), its first open-source code agent purpose-built for Lean 4 formal proof engineering—a niche but growing area at the intersection of formal methods and LLM-assisted software verification.

Why this matters

  • Voxtral TTS marks Mistral's entry into dedicated text-to-speech, an area where competitors like OpenAI and ElevenLabs have a head start.
  • Mistral Small 4's hybrid design—combining instruct, reasoning, and coding in one model with a 256k context window—positions it as a direct competitor to frontier models in versatility and context length.
  • Leanstral targets an increasingly active research community working on AI-assisted formal proofs using Lean 4, an area seeing growing investment from both academia and industry.
