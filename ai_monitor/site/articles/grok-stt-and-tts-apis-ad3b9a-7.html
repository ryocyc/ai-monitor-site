xAI Releases Grok Speech-to-Text and Text-to-Speech APIs

xAI has released two standalone audio APIs — Grok Speech to Text and Grok Text to Speech — built on the same technology powering Grok Voice in Tesla vehicles and Starlink customer support.

xAI Blog 2026-04-22 23:49 UTC Key: xai-blog::identity::grok-stt-and-tts-apis Confidence: strong Mode: claude

Article body

The Grok STT API handles both batch transcription and real-time streaming via REST and WebSocket endpoints, with features including word-level timestamps, speaker diarization, multichannel audio support, and Inverse Text Normalization that intelligently converts spoken numbers, dates, and currencies into structured output. The API claims strong performance across phone calls, meetings, video and podcasts, and general telephony, with an overall word error rate of 6.9% cited against four competitors on benchmark data. The TTS API supports fine-grained delivery control through inline speech tags like `[laugh]`, `[sigh]`, and `[whisper]`, and can generate speech in real time via WebSocket or as long-form audio via REST. Both products are positioned as standalone developer endpoints rather than Grok chatbot features, and are tied to the same underlying voice technology used in Tesla vehicles and Starlink customer support.

Why this matters

  • Grok STT batch pricing is $0.10 per hour — lower than most commercial speech transcription services, with an even lower streaming rate.
  • The API includes business-oriented features like entity recognition and Inverse Text Normalization out of the box, rather than requiring post-processing.
  • Multilingual support covers 25+ languages in a single API, with diarization for speaker identification across both recorded and live audio.
