AWS Security Agent and DevOps Agent Exit Preview, Now Generally Available

AWS has moved its Security Agent and DevOps Agent out of preview, making both frontier agents broadly available for autonomous security testing and cloud operations at scale.

AWS Machine Learning Blog 2026-04-23 17:08 UTC Key: aws-machine-learning-blog::identity::aws-machine-learning-blog-artificial-intellige Confidence: weak Mode: claude

Article body

AWS Security Agent, which handles on-demand penetration testing, and AWS DevOps Agent are now generally available. Both tools fall under a category AWS calls frontier agents—autonomous systems that work independently to achieve defined goals, scale across concurrent tasks, and run persistently for hours or days without needing constant human oversight.

Why this matters

  • Security teams gain an on-demand agent capable of autonomous penetration testing at a scale and duration previously requiring dedicated tooling or staffing.
  • DevOps teams can delegate multi-step cloud operations to an agent that persists across long-running workflows rather than timing out after short bursts.
  • Both agents represent the practical rollout of capabilities AWS previewed at its re:Invent conference, signaling that frontier agent architecture is moving from concept to production use.
