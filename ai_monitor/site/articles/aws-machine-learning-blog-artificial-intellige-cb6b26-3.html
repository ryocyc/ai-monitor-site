AWS Launches Frontier Agents for Security Testing and Cloud Operations

AWS Security Agent and DevOps Agent have exited preview and are now generally available, bringing autonomous AI agents to penetration testing and cloud operations tasks.

AWS Machine Learning Blog 2026-04-22 21:06 UTC Key: aws-machine-learning-blog::identity::aws-machine-learning-blog-artificial-intellige Confidence: moderate Mode: claude

Article body

AWS has made two frontier agents—AWS Security Agent and AWS DevOps Agent—generally available, marking a formal exit from their preview phase. These agents operate autonomously to achieve goals, scale across concurrent tasks, and run persistently for hours or days without constant human oversight, representing capabilities AWS announced at re:Invent under the "frontier agents" framing. Customers and partners who used the agents during preview reported that Security Agent helped uncover vulnerabilities across cloud environments.

Why this matters

  • Organizations can now run autonomous, on-demand penetration testing at scale through AWS Security Agent without needing to manually orchestrate each scan.
  • DevOps Agent brings similar autonomous capabilities to cloud operations, enabling AI-driven task execution across cloud environments for extended periods.
  • The GA marks the first time these frontier agent capabilities are officially supported for production use, having previously only been available in limited preview.
