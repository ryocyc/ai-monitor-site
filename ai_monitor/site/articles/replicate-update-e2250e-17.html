Replicate Launches Agent Skills for AI Coding Assistants

Replicate now publishes Agent Skills — markdown instruction files that give AI coding assistants expert guidance on model discovery, API execution, and prompting for image and video generation.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: strong Mode: claude

Article body

Replicate has published a collection of Agent Skills: markdown instruction files designed to give AI coding assistants structured, expert-level knowledge about working with its platform. The skills cover model discovery, model comparison, and API execution, along with detailed prompting techniques tailored to image generation and video generation models. Rather than leaving assistants to infer best practices, the files encode decisions about which models to choose, how to write effective prompts, and what tradeoffs to consider at each step.

The skills are installed via a single command — npx skills add replicate/skills — which pulls in all available instruction files and configures them for the user's coding assistant automatically. Replicate positions Skills as a complement to its existing MCP server: the MCP server hands an assistant the API tools to call, while Skills supply the knowledge about how to use those tools well. This separation means assistants can reason about model tradeoffs and prompting strategy even before a single API call is made.

The files follow the open Agent Skills specification and are compatible with Claude Code, OpenCode, OpenAI Codex, and other tools that implement the same spec. For developers building AI-assisted workflows, the release reduces the trial-and-error involved in getting an autonomous coding assistant to reliably use Replicate's model catalog.

Why this matters

  • Agent Skills turn a generic coding assistant into a domain expert on Replicate's model catalog, closing the gap between tool availability and tool know-how.
  • The open Agent Skills spec means the community — not just Replicate — can author and share instruction files for other platforms and use cases.
  • By covering the full pipeline from model selection to prompt construction, Skills make it realistic to run complex multi-step AI workflows with minimal manual guidance.
