AWS Security Agent and DevOps Agent Exit Preview, Now Generally Available

AWS has moved its Security Agent for on-demand penetration testing and its DevOps Agent out of preview and into general availability, marking a formal expansion of its frontier agent portfolio.

Article body

AWS has announced the general availability of two frontier agents: AWS Security Agent, which enables on-demand penetration testing, and AWS DevOps Agent. The company first introduced these autonomous systems under the 'frontier agents' banner at its re:Invent event. Both agents are designed to work independently to achieve goals, scale across concurrent tasks, and operate persistently over hours or days without sustained human oversight.

These agents represent a shift toward autonomous AI systems capable of handling complex, long-running workflows in cloud environments. The Security Agent specifically targets automated security testing, while the DevOps Agent is aimed at streamlining cloud operations at scale.

Early feedback from customers and partners who participated in the preview phase indicated that the Security Agent was effective at comprehensive security assessments. The move to general availability signals AWS's commitment to embedding autonomous AI deeper into enterprise cloud security and operations workflows.
