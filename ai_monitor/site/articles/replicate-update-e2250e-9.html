Replicate Publishes Agent Skills for Claude Code, OpenAI Codex, and OpenCode

Replicate has published a collection of markdown instruction files that give AI coding assistants expert-level knowledge about its platform, models, and API.

Replicate Changelog 2026-04-22 23:49 UTC Key: replicate-changelog::identity::replicate-update Confidence: moderate Mode: claude

Article body

Replicate now publishes agent skills — markdown instruction files that give coding assistants expert knowledge about working with AI models on the platform. According to the changelog, skills cover model discovery, model comparison, and API execution, alongside detailed prompting techniques for image generation and video generation models.

The skills follow the open Agent Skills spec and are designed to work with Claude Code, OpenCode, OpenAI Codex, and other compatible tools. Installation is handled via a single command: `npx skills add replicate/skills`, which pulls all of Replicate's skills into a project and configures them for the coding assistant automatically.

Replicate draws a clear distinction between skills and its existing MCP server. The changelog notes that MCP gives a coding assistant API tools — the means to make calls. Skills give it knowledge about how to use those tools well: which models to pick, how to write prompts, and what tradeoffs to consider. The two are described as complementary rather than redundant.

Why this matters

  • Agent skills make it practical to use a coding assistant for complex Replicate workflows without manually explaining platform conventions each session.
  • The open Agent Skills spec means the files are portable and could be adapted for other platforms adopting the same standard.
  • Skills and the MCP server together aim to cover both the "how" (tools) and the "why/when" (knowledge) of working with Replicate's model catalog.
