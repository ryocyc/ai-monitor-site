Mistral March Releases: Voxtral TTS, Mistral Small 4, Leanstral, Moderation 2603

Mistral shipped four new models in March 2026, spanning text-to-speech, a hybrid instruct-reasoning-coder, a Lean 4 proof agent, and an updated moderation system.

Mistral Changelog 2026-04-22 23:49 UTC Key: mistral-changelog::identity::mistral-24.09 Confidence: moderate Mode: claude

Article body

Mistral released Voxtral TTS (voxtral-tts-2603) on March 23, a text-to-speech model featuring zero-shot voice cloning, multilingual support, and real-time streaming. The addition marks Mistral's first dedicated audio generation model available through its API. Also on March 23, Mistral introduced Leanstral (labs-leanstral-2603), an open-source code agent purpose-built for Lean 4 formal proof engineering—the company's first publicly released tool in that domain.

On March 16, Mistral launched Mistral Small 4 (mistral-small-2603), a hybrid model that unifies instruct, reasoning, and coding capabilities in a single multimodal package with a 256,000-token context window. The model is available via API. Three days earlier, on March 12, the company released Mistral Moderation 2603 (mistral-moderation-2603), an updated content moderation model. Concurrent with the moderation release, Mistral extended Custom Guardrails support to Agents and Conversations, allowing the feature to be configured at the agent level via a parameter or passed per-request on both the /v1/conversations and /v1/chat/completions endpoints.

Why this matters

  • Mistral Small 4's hybrid design puts instruct, reasoning, and coding in one model with a 256k context window, signaling a consolidation trend in Mistral's model lineup.
  • Voxtral TTS gives Mistral users a path into real-time audio generation with voice cloning, expanding the platform well beyond text-only modalities.
  • Leanstral's open-source release makes Lean 4 formal proof assistance freely available through Mistral's ecosystem, a niche but growing area in AI-assisted software verification.
