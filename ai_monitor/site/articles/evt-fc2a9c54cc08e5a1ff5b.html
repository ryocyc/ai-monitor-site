Hugging Face Gives Gradio Spaces a Built-In Agent API Endpoint

Every Gradio Space on Hugging Face now automatically exposes a machine-readable /agents endpoint, letting AI coding agents like Claude Code, Codex, and Pi discover and call the Space's capabilities without any manual configuration.

Hugging Face Changelog <span data-utc="2026-04-27T05:54:04+00:00">2026-04-27 05:54 UTC</span> Key: evt-fc2a9c54cc08e5a1ff5b Confidence: moderate Mode: claude

Article body

Hugging Face has quietly enabled a new default feature across its entire Gradio Spaces platform: every hosted Space now automatically serves an /agents.md endpoint. That endpoint returns a structured, machine-readable description of the Space's available inputs, outputs, and behaviors—essentially a self-documenting API contract that AI agents can parse and act on directly.

The feature is designed to bridge the gap between human-facing demo interfaces and the programmatic interfaces that AI agents need. Rather than requiring Space creators to manually write OpenAPI specs or tool definitions, the platform generates a compatible description automatically. The changelog specifically calls out Claude Code, Codex, and Pi as examples of agents that can point themselves at a Space and begin using it without setup.

For developers building AI-powered applications, this removes a friction point that has slowed adoption of third-party Spaces in agentic workflows. Rather than inspecting a Gradio demo UI manually, an agent can read the /agents.md description, determine whether the Space fits its task, and invoke it programmatically—turning shared Spaces into composable tools in an agent's toolchain.

Why this matters

  • Gradio Spaces are among the most widely shared AI demo and tooling artifacts on the Hugging Face Hub, making this a platform-wide improvement to interoperability.
  • Agent builders and workflow orchestrators can now treat Gradio Spaces as discoverable, callable tools without requiring Space creators to publish separate API documentation or integration guides.
  • The feature reduces the manual overhead of integrating third-party Spaces into coding-agent pipelines, potentially expanding the practical utility of the thousands of public Spaces already hosted on the platform.
