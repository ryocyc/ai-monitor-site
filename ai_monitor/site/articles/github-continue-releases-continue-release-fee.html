Article body

Continue's VS Code extension landed version 1.3.38 on March 27, 2026, authored by contributor RomneyDa. The release includes two new features worth noting: users can now filter session history by workspace directory, and the extension now supports a .continue/configs directory for configuration management. A bug fix also ensures that config.yaml is created and populated when accessed.
